# 🚗 Car Sales Dashboard (Excel Project)
## 📊 Project Overview
This project is an interactive **Car Sales Dashboard** built using Microsoft Excel.  
It provides insights into car sales performance, helping users analyze trends, compare metrics, and make data-driven decisions.

---

## 🎯 Objectives
- Track overall car sales performance  
- Analyze sales by different categories (brand, model, region, etc.)  
- Identify top-performing products  
- Visualize key metrics in an easy-to-understand format  

---

## 📁 File Included
- `Car Dashboard.xlsx` → Main Excel dashboard file containing:
  - Raw data
  - Pivot tables
  - Charts & visualizations
  - Interactive dashboard

---

## 🛠️ Tools & Technologies
- Microsoft Excel  
  - Pivot Tables  
  - Pivot Charts  
  - Slicers & Filters  
  - Data Cleaning  

---

## 📈 Dashboard Features
- 📌 Total Sales Overview  
- 📌 Sales by Car Brand  
- 📌 Sales Trends Over Time  
- 📌 Regional Sales Distribution  
- 📌 Interactive Filters (Slicers)  

---

## 🧩 How to Use
1. Download or clone this repository  
2. Open `Car Dashboard.xlsx` in Microsoft Excel  
3. Use slicers and filters to explore the dashboard  
4. Analyze insights from charts and KPIs
